# قذائف الحق — Missiles of Truth

Interactive trilingual web app based on **"Qadhaaif al-Haqq"** (Missiles of Truth) by Sheikh Mohammed al-Ghazali (1917-1996).

Published by Dar Al-Qalam, 1991 — 8 chapters, 244 pages.

## Features

- **20 Truth Cards** organized by 8 chapters — each refutes a myth about Islam
- **Myth or Fact** interactive flip cards
- **Quiz**: "Who Wants to Be a Scholar?" with lifelines (50/50, hint, Quran reference)
- **XP & Badges** gamification system
- **Young Explorer / Teen Scholar** age modes
- **Trilingual**: Arabic, English, French
- **3 Themes**: Fire, Night, Steel
- **Audiobook Narrator** with karaoke, duo reading, sleep timer
- **PWA** ready

## Book Chapters

1. Rational Thinking vs Blind Imitation
2. Defending Monotheism
3. What Enemies Want from Islam
4. Islam and Social Reform
5. Responding to Doubts
6. Islamic Dawah and Corrupt Rulers
7. Countering Materialism & Atheism
8. No Religion Without Freedom
